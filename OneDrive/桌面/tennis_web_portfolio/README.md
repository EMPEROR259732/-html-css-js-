# 網球球友媒合平台 (Tennis Partner Matching Web)

## 📌 專案簡介
這是一個使用 **HTML、CSS、JavaScript** 開發的前端作品，模擬「球友媒合平台」的功能。  
使用者可以搜尋球友、查看球友資訊，並透過按鈕進行媒合申請。  
此作品展示了前端工程師在 **UI設計、互動邏輯、DOM操作** 上的能力。

---

## 🎨 功能特色
- 🔍 **即時搜尋篩選**：輸入關鍵字即可篩選球友列表  
- 🧑‍🤝‍🧑 **球友卡片展示**：姓名、地點、媒合按鈕  
- 🎛️ **互動按鈕**：點擊「應徵媒合」會跳出提示  
- 📱 **響應式排版**：使用 CSS Grid，自動調整卡片排列  
- 🧭 **導覽列**：球友媒合、我的球場、會員中心、後台管理

---

## 🛠 技術棧
- **HTML5** → 頁面結構  
- **CSS3 (Flexbox / Grid)** → 排版與樣式  
- **JavaScript (ES6)** → 搜尋篩選、事件綁定、DOM操作
